# Holetto
会員登録制の掲示板 / Webアプリケーション
## 実装機能
- ユーザー登録機能
- ログイン機能
- 画像アップデート機能
- カテゴリー作成機能
- スレッド一覧表示機能
- ページネーション機能
- コメント投稿機能
- コメント削除機能
- いいね機能
- マイページ表示機能
